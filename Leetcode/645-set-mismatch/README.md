<h2><a href="https://leetcode.com/problems/set-mismatch/">645. Set Mismatch</a></h2>
<img src='https://img.shields.io/badge/Difficulty-Easy-green' alt='Difficulty: Easy' />
<hr>

<p>
You are given an array <code>nums</code> representing numbers from <code>1</code> to <code>n</code> where one number is duplicated and one number is missing.
</p>

<p>
Return the duplicated number and the missing number.
</p>

<h3>Approach</h3>
<ul>
  <li>Use a frequency array or set.</li>
  <li>Find the number that appears twice (duplicate).</li>
  <li>Find the number missing from 1 to n.</li>
</ul>

<h3>Time Complexity</h3>
<p>O(n)</p>

<h3>Space Complexity</h3>
<p>O(n)</p>
