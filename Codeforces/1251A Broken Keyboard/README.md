<h2><a href="https://codeforces.com/contest/1251/problem/A">1251A. Broken Keyboard</a></h2>
<img src='https://img.shields.io/badge/Difficulty-1000-brightgreen' alt='Difficulty: 1000' />
<hr>

<p>You are given a string <code>s</code> consisting of lowercase English letters.  
Some characters in the string represent keys on a broken keyboard.  
The keyboard types <strong>"a"</strong> but the other keys output letters normally.</p>

<p>Your task is to determine whether the string <code>s</code> can be generated by typing on this broken keyboard.  
In other words, check whether each character other than <code>'a'</code> appears <strong>contiguously</strong> in <code>s</code>.</p>

<p>If the string can be generated this way, print <code>"YES"</code>; otherwise, print <code>"NO"</code>.</p>

---

### 🔹 Explanation

The string is valid if and only if **each distinct character (other than 'a') appears in one contiguous block**.

For example:
- `abba` → NO (because 'b' is not contiguous)
- `abc` → YES
- `aaba` → YES
- `baaaba` → NO

---

### 🔹 Example 1

**Input:**
```
5
abba
abc
aaba
baaaba
xyz
```

**Output:**
```
NO
YES
YES
NO
YES
```

---

###  Solution Approach

1. Iterate through the string while tracking characters seen.
2. If a character repeats after an interruption by a different character, return `"NO"`.
3. Ignore `'a'` — it can appear anywhere without breaking the rule.

**Detailed Steps:**
- Maintain a set of seen characters.
- For each position:
  - If the current character is same as previous, continue.
  - If different and already seen before → invalid.
  - Otherwise mark as seen.

---

### ⏱ Complexity

- Time: O(n) — one pass through the string  
- Space: O(26) — constant extra space
